## Asset classification and ownership

Only owner of the asset is always in the best position to understand the true value of the asset. 

> Protecting asset should always be cost-effective.

> Owners are ultimately accountable for ensuring their assets are classified and thus protected appropriately. 

> Asset classification is ongoing process.

![[AssetClassification.png]]

| Class   | Government   | Non-government           | Damage         |
| ------- | ------------ | ------------------------ | -------------- |
| Class 3 | Top Secret   | Confidential/Proprietary | Grave Damage   |
| Class 2 | Secret       | Private                  | Serious Damage |
| Class 1 | Confidential | Sensitive                | Damage         |
| Class 0 | Unclassified | Public                   | No Damage      |

### Asset classification process / steps

1. Asset inventory
2. Determine & assign ownership
3. Classify based on value
4. Protect & handle based on classification
5. Assess and review 

> Value of asset changes - due to age, or to legal, regulatory, or compliance needs... 

| Classification                               | Categorization                                 |
| -------------------------------------------- | ---------------------------------------------- |
| System of classes ordered according to value | The act of sorting into defined classification |

### Labeling vs Marking

Labeling results in output that is **system readable**
- Metadata 
- Barcodes 
- QR codes 
- RFID tags 
- GPS tags
Marking extends the intent of labeling in a way that can clearly be understood and executed **by humans**.

### Different types of owners exist:

- Hardware owners 
- Applications owners

Data Processor - Responsible for processing data **on behalf** of the owner/controller Ex: cloud provider

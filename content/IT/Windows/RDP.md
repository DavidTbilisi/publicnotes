Use the built-in Remote Desktop Connection (mstsc.exe) on Windows

### **Shadowing Sessions (Windows Server)**

- **What It Is**: You can view or control another user's active session without disconnecting them (useful for support or monitoring).
---
### **Multiple RDP Sessions on Windows 10**

- **What It Is**: By default, Windows 10 allows only one RDP session at a time.
- **Hidden Trick**: You can enable multiple concurrent sessions by modifying system files or using tools like **RDP Wrapper Library**.
    - **Warning**: This is unsupported by Microsoft and could violate licensing terms.
